# The Wedding Curator - Deploy Kit

This is the exact React landing page version, packaged for Vercel.

## Before deploying
1. Keep the logo file in `public/TWC-logo.jpeg`
2. Replace the Formspree placeholder in:
   `src/WeddingCuratorLandingPage.jsx`

Replace:
`https://formspree.io/f/your-form-id`

with your real Formspree endpoint.

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
- Push this folder to GitHub
- Import the repo into Vercel
- Choose Hobby plan
